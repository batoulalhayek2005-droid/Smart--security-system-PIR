تقرير مشروع نظام التحكم والأمان الأساسي
إعداد الطالبة: بتول الحايك إشراف الدكتور: عادل الكفري

1. مقدمة عن المشروع:
نظام أمان ذكي يعتمد على الحوسبة الطرفية (Edge Computing) باستخدام Raspberry Pi 5 للتمييز بين الحركة العابرة والاختراق المستمر.

2. معمارية النظام (Hardware):
المتحكم: Raspberry Pi 5.

الحساس: PIR Motion Sensor (GPIO 17).

المخرجات: LED (GPIO 18) و Buzzer (GPIO 22).

3. منطق الكود (Logic):
تم استخدام مكتبة gpiozero بلغة Python.

يعتمد النظام على دالة smooth_on للإضاءة التدريجية في حال كانت الحركة بسيطة.

يتم تفعيل panic_mode (وميض وصوت إنذار) إذا استمرت الحركة لأكثر من ثانيتين.

4. الخاتمة:
أثبت النظام قدرته على إدارة التداخلات الحسية بكفاءة عالية، وهو حجر أساس لإضافة تقنيات الرؤية الحاسوبية مستقبلاً
